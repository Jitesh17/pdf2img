## Installation
### Method 1: Pip install

```python
pip install pdf2img
```

### Method 2

```python
pip install https://github.com/jitesh17/pdf2img/archive/master.zip
```

### Method 3: Git clone

```python
git clone https://github.com/jitesh17/pdf2img.git
cd pdf2img
pip install -e .
```
